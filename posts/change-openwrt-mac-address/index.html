    Tutorial Mengganti Wireless MAC Address pada OpenWrt 21.02

    Jika ingin mengganti MAC address pada OpenWrt 21.02 ke atas karena kebutuhan tertentu, bisa dengan langkah-langkah berikut:

    1. Buka koneksi SSH ke router menggunakan PuTTY (biasanya: 192.168.1.1)

    2. Edit berkas pengaturan wireless dengan perintah berikut:

    vi /etc/config/wireless

    1. Untuk memulai mode input, tekan tombol i, perhatikan indikator di bagian pojok bawah kiri, akan berubah dari - menjadi I:

    2. Tambahkan opsi baris berikut di bagian config wifi-iface:

    option macaddr '<MAC address baru>'

    ⚠️ Bagian <MAC address baru> diganti dengan MAC address yang baru, sesuai format ini: C0:FF:EE:12:34:56

    1. Untuk menyimpan perubahan berkas, tekan tombol ESC, diikuti dengan mengetikkan :wq

    2. Restart wifi agar menggunakan pengaturan baru dengan perintah berikut:

    wifi

    1. Hasilnya bisa dilihat di menu Network -> Wireless
