    Solusi Kecepatan Upload Sangat Kecil / Tidak Semestinya di OpenWrt

    Router OpenWrt anda sudah terkoneksi dengan jaringan @wifi.id/WMS, tapi kecepatan upload ketika dites sangat kecil?

    Contoh:

    Kemungkinan besar solusinya adalah: set fragmentation threshold interface wifi ke 2346. Langkah-langkah detilnya bisa disimak di bawah:

    Langkah-langkah#

    1. Koneksikan perangkat (PC/laptop) ke router (biasanya 192.168.1.1)

    2. Masuk ke tab Network -> Wireless

    3. Tekan Edit di bagian wifi yang mau dirubah fragmentation threshold-nya

    4. Masuk ke tab Advanced Settings, lalu scroll ke bagian Fragmentation Threshold

    5. Isikan 2346 sebagai nilai Fragmentation Threshold, lalu tekan Save and Apply

    6. Silahkan dicoba tes kecepatan kembali, harusnya sekarang sudah normal kembali seperti ini:

    Bug fragmentation threshold ini ada di rilis OpenWrt 19.07 dan snapshot sampai tulisan ini diterbitkan. Bug hanya berlaku ketika router dihubungkan ke Access Point tertentu.
