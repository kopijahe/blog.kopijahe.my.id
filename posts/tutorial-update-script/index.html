Tutorial Update Script Autologin (v1/v2)

9 November 2020 · KopiJahe

Mengalami permasalahan di script autologin? Silahkan coba untuk update script-nya terlebih dahulu…

Berikut langkah-langkahnya:

  1. Buka PuTTY, koneksikan perangkat (PC/laptop) ke router melalui SSH (biasanya 192.168.1.1)

  2. Ketikkan perintah berikut, sesuaikan dengan script yang anda gunakan:

Varian Script Perintah
autologin curl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
