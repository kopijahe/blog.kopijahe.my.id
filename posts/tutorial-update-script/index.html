    Tutorial Update Script Autologin (v1/v2)  [draft]

    9 November 2020 · KopiJahe

    Mengalami permasalahan di script autologin? Silahkan coba untuk update script-nya terlebih dahulu…

    Berikut langkah-langkahnya:

    1. Buka PuTTY, koneksikan perangkat (PC/laptop) ke router melalui SSH (biasanya 192.168.1.1)

    2. Ketikkan perintah berikut, sesuaikan dengan script yang anda gunakan:

    Varian ScriptPerintah
    autologincurl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
    autologin-firefoxcurl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-firefox.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
    autologin-googlecurl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-google.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
    autologin-wmscurl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-wms.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
    autologin-wms-firefoxcurl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-wms-firefox.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
    autologin-wms-googlecurl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-wms-google.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
    autologin-v2curl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-v2.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
    autologin-firefox-v2curl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-firefox-v2.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
    autologin-google-v2curl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-google-v2.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh
    autologin-wms-v2curl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-wms-v2.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh

    autologin-wms-google-v2curl https://raw.githubusercontent.com/kopijahe/wifiid-openwrt/master/scripts/autologin-wms-v2.sh -o /etc/autologin.sh && chmod +x /etc/autologin.sh

    1. Reboot router dengan mengetikkan perintah: reboot && exit
