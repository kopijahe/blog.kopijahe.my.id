Tutorial Pembaruan Berkas Autologin

18 December 2020 Â· KopiJahe

Jika menggunakan script autologin, autologin v2 atau kopijahe (dengan menggunakan parameter -f ), dan ingin mengganti akun yang digunakan, berikut adalah langkah-langkah yang diperlukan:

  1. Buka PuTTY, koneksikan perangkat (PC/laptop) ke router melalui SSH (biasanya 192.168.1.1)

  1. Ketikkan perintah berikut, sesuaikan dengan script yang anda gunakan:
Varian autologin Perintah
autologin/autologin-v2 `pgrep -f autologin
KopiJahe kopijahe -k

  1. Jika masih dalam posisi terkoneksi dengan internet (masih login), silahkan logout dulu.

  2. Buka halaman login

  3. Tekan tombol F12, lalu masuk ke tab Network dan centang Preserve log

  4. Lakukan login seperti biasa, lalu cari baris check_login.php atau quarantine.php (misal untuk jaringan @wifi.id, wms dan turunannya, sesuaikan dengan jaringan anda)

  5. Klik kanan barisnya, lalu pilih Copy -> Copy as cURL (bash)

  6. Buka koneksi SSH ke router menggunakan PuTTY (biasanya: 192.168.1.1)

  7. Hapus berkas login lama dengan menggunakan perintah

rm <nama file>

Misal:

rm /etc/login_file.txt

  1. Ketikkan perintah berikut untuk membuat berkas baru yang akan digunakan untuk proses autologin:
vi <nama berkas>

Misal:

vi /etc/login_file.txt

  1. Untuk memulai mode input, tekan tombol i, perhatikan indikator di bagian pojok bawah kiri, akan berubah dari - menjadi I:

  1. Paste hasil dari no. 4 dengan menggunakan klik kanan:

  1. Hapus baris --compressed dan akhiran \ di berkas

  1. Untuk menyimpan perubahan berkas, tekan tombol ESC, diikuti dengan mengetikkan :wq

  1. Proses autologin bisa dijalankan kembali, bisa dengan cara reboot router, cabut-pasang kabel power, atau dengan perintah:
sh /etc/rc.local
